Описание
GUI-приложение "Random Task Generator" для генерации случайных задач из предопределенного списка с возможностью добавления собственных задач, фильтрации по категориям и сохранения истории в JSON формате.

Требования
Python 3.8+
Библиотеки: tkinter, json, random
Установка и запуск
Клонируйте репозиторий:
git clone https://github.com/ваш-username/random-task-generator.git
cd random-task-generator

Запустите приложение:
python task_generator.py

Функциональность
Генерация случайных задач из предопределенного списка
Добавление пользовательских задач с валидацией
Фильтрация задач по категориям (учёба, спорт, работа)
Сохранение и загрузка истории задач в JSON формате
Отображение истории сгенерированных задач
Структура проекта
random-task-generator/
├── task_generator.py      # Основной файл приложения
├── tasks.json            # Файл для хранения истории задач
├── .gitignore           # Git ignore файл
└── README.md           # Этот файл

Примеры использования
Генерация случайной задачи: Нажмите кнопку "Сгенерировать задачу"
Добавление своей задачи: Введите задачу в поле и нажмите "Добавить задачу"
Фильтрация: Выберите категорию из выпадающего списка
Сохранение истории: История автоматически сохраняется при закрытии приложения
Тестирование
Позитивные тесты:
Генерация существующих задач
Добавление корректных задач
Фильтрация по всем категориям
Негативные тесты:
Попытка добавить пустую задачу
Попытка добавить задачу из только пробелов
Граничные случаи:
