# Movie Library - Личная кинотека

## 📌 Описание проекта

**Movie Library** - это десктопное приложение на Python для управления личной коллекцией фильмов. Приложение позволяет добавлять, просматривать, редактировать и удалять записи о фильмах с удобным графическим интерфейсом.

## ✨ Основные возможности

- 📁 **Хранение информации о фильмах:**
  - Название фильма
  - Режиссер
  - Год выпуска
  - Жанр
  - Рейтинг (0-10)
  - Дата добавления
  - Заметки

- 🔍 **Поиск и фильтрация:**
  - Поиск по названию
  - Фильтрация по жанру
  - Фильтрация по году выпуска
  - Сортировка по рейтингу

- 📊 **Статистика:**
  - Общее количество фильмов
  - Средний рейтиг коллекции
  - Количество фильмов по жанрам
  - Графики статистики

## 🛠 Технологии

- **Python 3.8+**
- **Tkinter** - для графического интерфейса
- **JSON** - для хранения данных
- **Matplotlib** - для построения графиков (опционально)
